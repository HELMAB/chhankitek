Usage Examples

Two simple ways to convert dates

use HasChhankitek;

$date = $this->chhankiteck(
    CarbonImmutable::now()
        ->setTimezone('Asia/Phnom_Penh')
);

// Get components
$date->getDayOfWeek();
$date->getLunarDay();
$date->getLunarMonth();
$date->getLunarZodiac();
$date->getLunarEra();
$date->getLunarYear();

Using the Trait

Include the HasChhankitek trait in your Laravel controllers or models. This approach gives you access to individual date components and provides fine-grained control over the lunar calendar data.

Access individual components
Perfect for custom formatting
Use in controllers and models

Using the Helper Function

Use the global toLunarDate() helper for quick conversions anywhere in your application. This is the simplest approach when you need the complete formatted lunar date string.

Available globally
Get formatted string instantly
Perfect for Blade templates 
$lunarDate = toLunarDate(
    CarbonImmutable::now()
        ->setTimezone('Asia/Phnom_Penh')
);

// Get full string
